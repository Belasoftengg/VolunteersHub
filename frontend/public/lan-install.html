Install VolunteerHub as a PWA

You're accessing VolunteerHub over a LAN IP address. To install the app on your device, follow the instructions below for your specific device.

Why am I seeing this page?

Progressive Web Apps (PWAs) require either HTTPS or localhost for full functionality. When accessing over a LAN IP address, some PWA features like automatic installation prompts may not work. This page provides manual installation instructions instead.

Install on iOS (iPhone/iPad)

  1. Open Safari (PWA installation doesn't work in other browsers on iOS)
  2. Tap the Share button
  3. Scroll down and tap "Add to Home Screen"
  4. Tap "Add" in the top right corner

Install on Android

  1. Open Chrome (recommended) or another supported browser
  2. Tap the menu button (⋮) in the top right
  3. Tap "Install app" or "Add to Home screen"
  4. Follow the on-screen instructions

Install on Desktop (Chrome, Edge, etc.)

  1. Look for the install icon in the address bar (usually on the right side)
  2. Click it and follow the prompts
  3. If you don't see an install icon, click the menu button (⋮) and look for "Install VolunteerHub"

For Developers

For the best PWA development experience:

  • Use localhost instead of LAN IP when possible
  • Set up HTTPS for your development server
  • Test production builds for full PWA functionality
